    A propos

    Apprenez, codez, optimisez et créez des applications web modernes
    formation efficace vous permettra de maîtriser les langages et technologies utiles pour un développeur back-end. Un parcours mélangeant autonomie, pratique et accompagnement sur mesure

    VISIO CONFÉRENCE

    Des séances en live, face à face, avec un formateur expert, qui personnalise votre apprentissage, critique votre travail, vous coache pour vous faire avancer !

    PROJET PRO

    En fin de formation, vous préparez un projet professionnel, de manière autonome, de manière à mettre en pratique les notions acquises pendant votre cursus.

    CERTIFICAT DE COMPLÉTION

    Une fois la formation suivie, les tests passés, le projet pro validé, vous obtenez un certificat que vous pourrez fiérement abordé. Ce dernier atteste de vos nouvelles compétences.

    Formations

    Le Développement Web de A à Z. Apprenez HTML, CSS, Javascript, jQuery, Bootstrap, PHP, MySQL, Wordpress...

    HTML

    La formation HTML s’articule autour de plusieurs axes : découvrir le HTML, le comprendre et l’intégrer. Chaque jour, vous découvrez de nouveaux enseignements que vous mettez aussitôt en œuvre. Notre formation HTML repose à 90% sur la pratique
    Prix : 200dt

    CSS

    Les CSS (Cascading Style Sheets ou feuilles de styles en cascade) permettent d’appliquer des styles et des éléments de mise en forme afin de créer des sites Web attractifs. C’est pourquoi nous vous proposons une formation
    Prix : 250dt

    JavaScript

    Notre formation JavaScript est ouverte à tous : aucune expérience en développement n’est nécessaire, pas plus qu’un diplôme. Nous vous accompagnons de A à Z. Notre méthodologie adaptée vous donne l’opportunité de devenir développeur Web
    Prix : 200dt

    jQuery

    La formation jQuery vous permettra de maîtriser l’une des bibliothèques JavaScript les plus utilisées. A l’issue de la formation vous serez à même d’utiliser jQuery efficacement pour écrire vos scripts coté client.
    Prix : 300dt

    Bootstrap

    Une formation Bootstrap vous permettra de découvrir et maîtriser toutes les spécificités de ce framework indispensable pour développer des sites et applications Web en responsive design. Grâce à notre formation basée à 90 % sur la pratique
    Prix : 250dt

    wordpress

    Cette formation se repose sur WORDPRESS , l’un des systèmes de gestion de contenu primé (CMS) avec une facilité d’utilisation et une extensibilité, orienté vers la création des blog s’adresse aux développeurs de site web.
    Prix : 200dt

    Programme

    voici notre programme html de 21 Fev 2019

    Jour1 HTML

    Les bases du HTML

    Les balises et leurs attributs. Le HTML et le HTML 5. Les navigateurs, les règles importantes à respecter.
    Jour2 HTML

    Créer une page HTML

    Page, en-tête, titre, commentaires, langue. Police, taille, gras, italique, caractères spéciaux. Paragraphe, alignement, retrait, listes à puces, blocs de citation.
    Jour3 HTML

    Insérer des médias dans une page

    Images : gif, jpg, png. Taille, positionnement, texte alternatif. Son, video, animation.
    Jour4 HTML

    Définir la navigation et les liens hypertextes

    Lien vers une ancre, une page du site, une url externe, vers une messagerie. Lien sur une image, une partie d'image.
